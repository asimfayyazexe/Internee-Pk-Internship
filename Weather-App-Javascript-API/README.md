# Weather-App-Javascript-Api

A simple weather app built with **HTML, CSS, and JavaScript** that lets you check the current weather for any city in the world. It shows temperature, humidity, wind speed, and a weather icon in a clean card layout.

---

## Features

⭕ **Search by City** – Type any city and get the weather instantly.
🌡️ **Weather Details** – Displays temperature, humidity, and wind speed.
☁️ **Weather Icons** – Shows icons for common conditions like Clear, Clouds, Rain, Mist, and Snow.
❌ **Error Handling** – Shows “City not found” if the city doesn’t exist.
📝 **Simple Design** – Easy-to-read layout without unnecessary complexity.

---

## Technologies Used

* **HTML5** – Structure of the page
* **CSS3** – Styling for card and input elements
* **JavaScript** – Fetching data from OpenWeatherMap API and updating the page
* **OpenWeatherMap API** – Real-time weather data
* **Font Awesome** – Search icon

---

## How to Use

1. Open `index.html` in your browser.
2. Type a city name in the search box.
3. Click the **Search** button.
4. View the temperature, humidity, wind speed, and weather icon.
5. If the city is not found, an error message will appear.

---

## Notes

* Replace `YOUR_API_KEY` in `script.js` with your own OpenWeatherMap API key.
* Weather icons are stored in the `images` folder.
* Designed to be simple and easy to use.
